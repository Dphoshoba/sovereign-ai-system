# EV-KOS Enterprise Review Board

## Purpose

This document defines the Enterprise Review Board contract for EA-5.

## Seats

- Enterprise Administration Seat.
- Enterprise Reviewer Seat.
- Research Lead Seat.
- Content Strategy Seat.

## Voting Mode

All seats are advisory-only in EA-5. No seat can approve execution.

## Persistence

The review board is not persisted and does not modify users, roles, memberships, approvals, audit records, graph records, or database state.
