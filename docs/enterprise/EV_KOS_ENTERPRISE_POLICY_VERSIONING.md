# EV-KOS Enterprise Policy Versioning

## Purpose

EA-6 defines version metadata for enterprise policy contracts.

## Version Contract

Each policy version includes:

- Policy id.
- Version label.
- State.
- Change summary.
- Superseded version.
- Non-persistence flag.
- Non-execution flag.

## Current State

All EA-6 versions are `review-required` and `1.0-preview`.
