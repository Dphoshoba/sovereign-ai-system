# EV-KOS Enterprise Beta Provider Rollout (EB-7)

EB-7 defines provider rollout planning checkpoints in report-only mode.

## Rollout Stages

- Selection freeze.
- Sandbox contract.
- Claim contract.
- Tenant pilot plan.
- Cutover gate.

## EB-7 Boundaries

- No provider integration.
- No provider installation.
- No runtime auth activation.
- No sessions or JWT.
- No middleware changes.
