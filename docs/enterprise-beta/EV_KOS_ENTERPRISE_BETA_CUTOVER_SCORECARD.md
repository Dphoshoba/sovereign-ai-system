# EV-KOS Enterprise Beta Cutover Scorecard (EB-9)

EB-9 defines a report-only cutover scorecard for guarded activation rehearsals.

## Scorecard Items

- Canary success threshold.
- Rollback SLA threshold.
- Approval gate threshold.
- Tenant impact threshold.

## Rule

If scorecard thresholds are not met, the plan remains no-go and activation stays
disabled.
