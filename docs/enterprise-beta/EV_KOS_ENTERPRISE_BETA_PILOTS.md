# EV-KOS Enterprise Beta Tenant-Limited Activation Pilots (EB-10)

EB-10 defines report-only tenant-limited activation pilots and readiness gates.

## Pilot Scope

- Pilot cohort design.
- Tenant and activation wave design.
- Operator holdpoint design.
- Guardrail telemetry design.
- Rollback drill design.

No runtime activation occurs in EB-10.
