# EV-KOS Enterprise Beta Quotas

EB-4 defines tenant and operator quota contracts.

## Tenant Quotas

Tenant quotas describe preview requests, intent packages, and review packages
per day by tenant tier.

## Operator Quotas

Operator quotas describe preview actions, package actions, and high-risk action
limits by role.

## Enforcement

Quota enforcement is not active. Overrides must require future approval.
