# EV-KOS Enterprise Beta Guard Decisions

EB-3 defines the decision contract that future route guards will use.

## Decision Outcomes

- `ALLOW_PREVIEW`
- `REQUIRES_REVIEW`
- `BLOCK_RUNTIME`
- `MISSING_CONTEXT`

These outcomes do not enforce runtime behavior yet. They are planning contracts
for future guard implementation.

## Required Decision Context

- Route pattern.
- Required claims.
- Provided claims.
- Missing claims.
- Required permissions.
- Operator explanation.
- Runtime enforcement flag.
