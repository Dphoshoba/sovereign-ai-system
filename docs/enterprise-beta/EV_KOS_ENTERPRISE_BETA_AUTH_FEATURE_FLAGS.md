# EV-KOS Enterprise Beta Auth Feature Flags (EB-8)

EB-8 defines report-only feature-flag boundaries for auth dry-run governance.

## Boundaries

- `auth_dry_run_enabled`
- `auth_claim_shadow_validation`
- `auth_tenant_cutover_simulation`
- `auth_rollback_simulation`

All flags are default-disabled in EB-8 and remain non-runtime.
