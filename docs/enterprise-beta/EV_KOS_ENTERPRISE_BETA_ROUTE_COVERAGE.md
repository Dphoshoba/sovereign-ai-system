# EV-KOS Enterprise Beta Route Coverage

EB-2 measures report-only route coverage across high-risk API groups.

## Covered Areas

- Enterprise Beta APIs.
- Operator APIs.
- Ontology and graph readiness APIs.
- Content and publishing APIs.
- Research mission APIs.

## Known Gap

Legacy execution, publishing, and social routes require per-route guard mapping
before any runtime execution or publishing capability is enabled.
